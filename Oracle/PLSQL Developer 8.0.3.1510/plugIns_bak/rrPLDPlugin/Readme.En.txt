================================================================================
rrProject Plugin for PL/SQL Developer
================================================================================

rrProject Plugin for PL/SQL Developer - is a freeware toolset aimed to increase
productivity of PL/SQL Developer users.


Send your feedback and bug reports via E-mail:
Starina.ar@gmail.com.

Any donations will promote further plugin development:
https://www.paypal.com/cgi-bin/webscr?cmd=_s-xclick&hosted_button_id=W5QSW728PAN7L


Package contents:
  rrPLDPlugin.dll                    - plugin
  rrPLDPlugin\                       - plugin resources folder
     CnPack.Team.En.txt              - CnPack Team license agreement (english)
     License.En.txt                  - license agreement (english)
     License.Ru.txt                  - license agreement (russian)
     Readme.En.txt                   - brief description (english)
     Readme.Ru.txt                   - brief description (russian)
     rrWizRes.dll                    - tool icons
     Whats.New.En.txt                - changelog (english)
     Whats.New.Ru.txt                - changelog (russian)
  rrPLDPlugin\Data\                  - folder with default plugin settings
     rrPLDPlugin.ini                 - settings file
     DefaultDeactive.txt             - inactive tools list
     JavaDocComment.xml              - settings file
     MenuIcons.txt                   - settings file
     MenuIcons.dll                   - settings file
     UserSymbols.xml                 - settings file
     XmlComment.xml                  - settings file
  rrPLDPlugin\User\                  - folder with user plugin settings


================================================================================
Installation
================================================================================
rrProject Plugin for PL/SQL Developer supports PL/SQL Developer v7.0 and above.

Unzip archived package to PL/SQL Developer's Plugins subfolder.
