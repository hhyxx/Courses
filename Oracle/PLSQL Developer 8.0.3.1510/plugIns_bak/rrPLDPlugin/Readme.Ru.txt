================================================================================
rrProject Plugin for PL/SQL Developer
================================================================================

rrProject Plugin for PL/SQL Developer - это свободнораспространяемый набор
инструментов, предназначенный для увеличения производительности работы
пользователей в PL/SQL Developer.

Свои отзывы, предложения и отчеты об ошибка при работе плагина Вы можете
направлять на электронный адрес: Starina.ar@gmail.com.

Любая финансовая помощь будет способствовать дальнейшей разработке:
https://www.paypal.com/cgi-bin/webscr?cmd=_s-xclick&hosted_button_id=W5QSW728PAN7L


Содержимое архива:
  rrPLDPlugin.dll                    - плагин
  rrPLDPlugin\                       - папка с данными плагина
     CnPack.Team.En.txt              - лицензия CnPack Team (английский)
     License.En.txt                  - лицензия (английский)
     License.Ru.txt                  - лицензия (русский)
     Readme.En.txt                   - краткое описание (английский)
     Readme.Ru.txt                   - краткое описание (русский)
     rrWizRes.dll                    - иконки инструментов
     Whats.New.En.txt                - история (английский)
     Whats.New.Ru.txt                - история (русский)
  rrPLDPlugin\Data\                  - папка с данными по умолчанию
     rrPLDPlugin.ini                 - файл настройки
     DefaultDeactive.txt             - список неактивных инструментов
     JavaDocComment.xml              - файл настройки
     MenuIcons.txt                   - файл настройки
     MenuIcons.dll                   - файл настройки
     UserSymbols.xml                 - файл настройки
     XmlComment.xml                  - файл настройки
  rrPLDPlugin\User\                  - папка с данными пользователя


================================================================================
Установка
================================================================================
rrProject Plugin for PL/SQL Developer поддерживает PL/SQL Developer версии
7.0 и выше.

Распакуйте архив в папку Plugins, где установлен PL/SQL Developer.
